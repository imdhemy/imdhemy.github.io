# Skill-Specific Naming Notes

Use the post slug as the base filename.

Examples:

- `2023-04-16-hidden-gems-in-composer.md` -> `hidden-gems-in-composer.jpg`
- `2026-03-13-currying-in-javascript.md` -> `currying-in-javascript.png`

Keep the actual source image extension unless the user explicitly asks for format conversion.

For this skill, "SEO-friendly" means:

- lowercase
- hyphen-separated
- no date prefix
- no random camera or export names
- derived from the post slug whenever possible
