# Prompt Template

Fill this template from the target post:

```text
Create a minimal cartoon-style illustration that matches a sketchnote visual identity for a personal tech blog.

Style:
- Hand-drawn line-based illustration (same line style as sketchnotes)
- Slightly imperfect black marker outlines
- Characters are simple, rounded, expressive but minimal
- Use light color fills only, like soft marker coloring inside the lines
- Do NOT use fully rendered or polished cartoon shading
- Keep it closer to "colored sketch" than "full cartoon"

Color:
- Limited palette only
- Use same pastel colors as sketchnote style (light blue, soft orange, grey, soft green)
- Use deep teal (#0F766E) as primary accent
- Flat color only, no gradients
- Minimal or no shadows

Background:
- Subtle dotted notebook paper pattern (same as sketchnote style)
- Very light and clean
- No plain white background

Composition:
- Focus on ONE clear idea only
- Maximum 3 characters
- Maximum 1 main object (e.g. pipeline)
- Maximum 2–3 supporting elements
- Large clear focal point
- Plenty of breathing space
- No clutter, no dense details
- Landscape layout (1200x675)

Tone:
- Slightly playful but professional
- Calm, not chaotic
- Easy to understand in 2 seconds

Output requirements:
- Exact size: 1200x675 pixels (16:9)
- Optimized for web
- Avoid micro-details and visual noise

Avoid:
- Overcrowded scenes
- Too many icons or labels
- Complex environments
- AI-stock illustration look
- Highly detailed faces or textures

Subject:
[INSERT CORE IDEA ONLY]

Scene:
[DESCRIBE ONLY 1 CLEAR SCENE]
```
