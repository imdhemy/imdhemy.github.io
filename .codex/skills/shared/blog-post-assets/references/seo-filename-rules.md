# SEO Filename Rules

Use the post slug as the canonical base name.

## Slug Extraction

For a dated post filename such as:

- `2023-04-16-hidden-gems-in-composer.md`

use:

- `hidden-gems-in-composer`

For non-dated files, use the filename stem directly.

## Filename Shape

- lowercase only
- words separated by hyphens
- no spaces
- no date prefix in the asset filename
- no generic upload names such as `image-1`, `IMG_2048`, or `final-final`

## Extension Rule

Keep the source image extension:

- `.png`
- `.jpg`
- `.jpeg`
- `.webp`

Do not convert formats unless the user explicitly asks for it.
