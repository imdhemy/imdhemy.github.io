# Shared Visual Rules

These rules apply to both blog illustration skills unless the style-specific skill says otherwise.

## Brand Identity

- The image should feel like a software engineer explaining an idea visually.
- The tone is clean, smart, slightly playful, and still professional.
- Keep the visual language consistent across the blog.

## Shared Look

- Use slightly imperfect black marker outlines.
- Use a very light dotted notebook paper pattern in the background.
- Never leave the background as a plain empty white field.
- Use a limited pastel palette:
  - light blue
  - soft orange
  - light grey
  - soft green
- For the cartoon + sketchnote style, deep teal `#0F766E` is the primary accent.
- Keep the image in landscape format at exactly `1200x675`.
- Optimize for web use and avoid unnecessary micro-details.

## Avoid

- photorealism
- gradients
- heavy shadows
- corporate flat illustration style
- AI stock illustration look
- dense clutter
- overrendered textures

## Consistency Rule

When choosing between a clever visual and a readable one, prefer readability. The blog identity should feel coherent from post to post.
