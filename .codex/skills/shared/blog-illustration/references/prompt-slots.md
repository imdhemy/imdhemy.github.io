# Prompt Slot Rules

Use concise slot values. Fill them from the post instead of repeating the article title mechanically.

## Subject

Use for the sketchnote skill.

- State the main concept the image explains.
- Phrase it as a visual idea, not a chapter title.
- Prefer one concrete sentence fragment.

## Supporting Elements

Use for the sketchnote skill.

- Include only 2 to 5 supporting doodles or icons.
- Choose items that help explain the central concept.
- Avoid long lists.

## Core Idea

Use for the cartoon + sketchnote skill.

- State the one idea the scene must communicate.
- Keep it short and direct.

## Scene

Use for the cartoon + sketchnote skill.

- Describe only one clear scene.
- Include at most:
  - 3 characters
  - 1 main object
  - 2 to 3 supporting elements

## Output Shape

Return the final prompt with the placeholders fully replaced. Do not include analysis notes, option lists, or alternative prompts unless the user asks for variants.
