# Post Analysis

Read the full target post unless the user explicitly names a narrow section to illustrate.

## Find The Core Idea

Extract these signals:

- the main teaching point
- the emotional tone
- the strongest metaphor already present in the writing
- the domain objects that recur in the post
- what a reader should understand within two seconds from the image

Prefer ideas that are clearly supported by the post text. Do not invent unrelated metaphors just because they are visually convenient.

## Thumbnail Extraction

For a thumbnail:

- choose the article-wide concept
- compress the article into one central visual idea
- pick only a small number of supporting elements

Good thumbnail inputs are:

- a repeated metaphor from the article
- a key workflow or failure mode
- a strong contrast or tension inside the post

## Inline Illustration Extraction

For an inline illustration:

- focus on one purpose only
- if the user gives a section heading, prioritize that section
- if the user gives a purpose statement, use that as the scope boundary
- ignore unrelated parts of the article

The result should teach one thing, not summarize the whole article.
